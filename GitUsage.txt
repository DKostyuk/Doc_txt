Создать Репозиторий на github (лучше с названием проекта)
git bush here - открыть консоль в нужной папке (в папке проекта)
git init . -создать локальную папку репозитория на компьютере
надо создать файл в корне проекта .gitignore - указывает файлы, которые НЕНАДО выкладывать - рабочие файлы
git add . - Добавить все фалы,  а не по одному
git commit -a -m "any comments about this commit" - все файлы которые отслеживают GITмы зафиксируем для возможности возврата в будущем
Для первого раза (url - адрес проекта написано после создания репозитория на Github):
git remote add origin https://github.com/DKostyuk/renew.git
git remote -v   --- посмотреть что у нас зарегистрировано
git push origin master - когда мы хоти все файлы которые есть в локальной git папке  и они прошли commit >>> хотим отправить на сервер
где origin - название сервера (в нашем примере https://github.com/DKostyuk/renew.git)
	master - название ветки, по у молчанию master - главная ветка проекта
git pull origin master - скачиваем все что есть на Github
